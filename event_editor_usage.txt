[CHRONO TAPE] 이벤트 에디터 사용 설명서

1. 실행 방법
   - 프로젝트 폴더 내의 'event_editor.html' 파일을 더블 클릭하여 브라우저(Chrome 권장)로 엽니다.

2. 기본 정보 입력
   - 이벤트 ID: 고유한 영문 식별자입니다. (예: meeting_witch, old_ruins)
   - 제목: 게임 내 팝업 상단에 표시될 제목입니다.
   - 내용: 상황을 설명하는 텍스트입니다. 줄바꿈을 포함할 수 있습니다.
   - 등장 조건 (req): 이 이벤트가 등장하기 위한 조건을 자바스크립트 코드로 적습니다.
     * data 변수를 통해 게임 데이터에 접근합니다.
     * 예시:
       - data.currentHp < 50      (체력이 50 미만일 때만)
       - data.gold >= 200         (골드가 200 이상일 때만)
       - data.stage >= 2          (2스테이지 이상일 때만)
     * 비워두면 조건 없이 랜덤하게 등장합니다.

3. 선택지 관리
   - [+ 선택지 추가] 버튼을 눌러 선택지를 늘릴 수 있습니다.
   - 버튼 텍스트: 유저가 클릭할 버튼에 적힐 내용입니다.
   - 선택 조건: 해당 선택지를 클릭하기 위한 조건입니다. (조건 불충족 시 버튼 비활성화)
     * 예시: data.artifacts.includes('key') (열쇠 유물이 있어야 가능)
   - 효과 스크립트: 선택 시 실행될 로직입니다. 반드시 결과 메시지(String)를 return해야 합니다.
     * 예시:
       data.gold -= 50;
       data.currentHp += 30;
       return "치료사를 고용하여 상처를 치료했습니다. (골드 -50, 체력 +30)";

4. 코드 적용 방법
   1) 모든 내용을 작성한 후 하단의 [💾 코드 생성] 버튼을 누릅니다.
   2) 우측 패널에 생성된 코드가 나타납니다.
   3) [📋 클립보드 복사] 버튼을 누릅니다.
   4) VS Code 등의 에디터로 'js/data/EventData.js' 파일을 엽니다.
   5) GAME_EVENTS 배열 안에 복사한 코드를 붙여넣습니다. (쉼표 , 주의)
      * 기존 이벤트 뒤에 콤마(,)를 찍고 붙여넣으세요.

5. 주의 사항
   - 작은 따옴표(')나 큰 따옴표(") 사용 시 문법 오류가 나지 않도록 주의하세요.
   - 효과 스크립트에서 return 문이 없으면 결과창에 undefined가 뜹니다.
